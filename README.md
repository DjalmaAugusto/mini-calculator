## Mini Calculator API

API Rest for a mini calculator

## Software Requirements

-   Node.js **12.3.0+**
-   MongoDB **4.4.1+**
-   Redis   **6.0.9+**

## How to install

### Clone GitHub repo:

```
git clone https://github.com/DjalmaAugusto/mini-calculator-api.git
```

### Install Dependencies

```
cd mini-calculator
npm install
```
